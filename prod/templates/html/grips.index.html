Break The Web Forward

01.01.2018 International Day

Before Y2k, the whole software world rallied together ahead of a specific date (Jan 1, 2000) to find and fix date bugs.

I hereby declare Jan 1, 2018 a similar impending "day of doom", or perhaps the first in a series of them, for all legacy bugs (backwards-compat, deprecated by standards) in the web platform technologies.

On this date, browsers will automatically "flip the switch", turning on fixes for existing legacy technical debt in JS, CSS, and HTML. After the day passes, browsers will remove the old legacy stuff that's no longer needed.

Let's break the web forward by intentionally and carefully breaking backwards compat.

As a community, we have plenty of time to rally volunteers together to help fix every single web property, public or private, maintained or not, that wants to move forward on the web. Otherwise, those forgotten parts of the web will be... well, forgotten.

Follow the progress of this movement on @BreakWebForward.

Let's Break The Web Forward!
